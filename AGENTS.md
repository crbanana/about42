# HYPERISE - сайт про 5opka

Веб-сайт со историями, новостями, лонгридами, статьями, заметками по стримера Пятерка

## Алгоритм работы

1. Получаем список видео с каналов нарезчиков (например https://www.youtube.com/@LigaKubizma)
2. Скачиваем транскрипции еще не обработанных видео
3. Первый агент обрабатывает транскрипции и добаляет их в структурированное вики
4. Вики добавляется в RAG
5. Второй агент придумывает идею (у них есть доступ к векторной базе знаний)
6. Третий агент пишет материал, который выкладывается на сайт

## Реализация

1. RAG находится в базе данных Neon
2. Статичный сайт на Render
3. Ежедневные запуски обновлений через Github Actions
4. Вики хранится в репозитории и обновляется первым агентом
5. Старницы сайта хранятся в репозитории и обновляются коммитами

## Правила разработки

1. Смотри скиллы GitHub, Render, Neon, LangCahin
2. Читай документацию
3. Делай все в serverless подходе и так, чтобы было 100% бесплатно
